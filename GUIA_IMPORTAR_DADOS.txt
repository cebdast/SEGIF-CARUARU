Este guia explica como importar dados de despesas e receitas no sistema.

--------------------------------------------------------------------------------
ACESSO À PÁGINA DE IMPORTAÇÃO
--------------------------------------------------------------------------------

1. No menu lateral, clique em "Banco de Dados"
2. Clique em "Importar Dados"

--------------------------------------------------------------------------------
COMO IMPORTAR
--------------------------------------------------------------------------------

1. SELECIONE O TIPO DE DADOS
   - Clique em "Importar Despesas" (verde) para dados de despesas
   - Clique em "Importar Receitas" (azul) para dados de receitas

2. ENVIE O ARQUIVO EXCEL
   - Arraste o arquivo para a área indicada, OU
   - Clique na área para selecionar o arquivo

3. CONFIRA A PRÉ-VISUALIZAÇÃO
   - O sistema mostrará os primeiros registros
   - Verifique se os dados estão corretos

4. CONFIRME A IMPORTAÇÃO
   - Clique em "Confirmar Importação"
   - Aguarde a mensagem de sucesso

--------------------------------------------------------------------------------
FORMATO DO ARQUIVO EXCEL
--------------------------------------------------------------------------------

O sistema aceita arquivos Excel (.xlsx) com MÚLTIPLAS ABAS.
Cada aba é identificada automaticamente pelo nome:

  Nome da Aba (contém)    |  Tipo de Dado
  ------------------------|---------------------------
  "empenhad"              |  Despesas Empenhadas
  "liquidad"              |  Despesas Liquidadas
  "pago" ou "pagamento"   |  Despesas Pagas
  "a pagar" ou "pagar"    |  Despesas A Pagar
  "retid" ou "retenção"   |  Retenções

IMPORTANTE: O nome da aba deve CONTER uma dessas palavras.
Exemplos válidos: "Empenhados 2024", "Despesas Liquidadas", "Pagamentos"

--------------------------------------------------------------------------------
COLUNAS ESPERADAS
--------------------------------------------------------------------------------

DESPESAS EMPENHADAS:
  - Nr emp. (número do empenho)
  - Data
  - Credor/Fornecedor
  - Espécie (Ordinário, Global, Estimativa)
  - Unidade orçamentária
  - Despesa
  - Fonte de recursos
  - Tipo
  - Função
  - Valor (R$)

DESPESAS LIQUIDADAS:
  - Seq. liq. (sequência da liquidação)
  - Data
  - Nr emp. (número do empenho)
  - Credor/Fornecedor
  - Unidade orçamentária
  - Função
  - Tipo
  - Valor (R$)
  - TOTAL (retenções)

DESPESAS PAGAS:
  - Nr Emp.
  - Seq. Liq.
  - Data
  - Credor/Fornecedor
  - Unidade Orçamentária
  - Função
  - Tipo
  - Valor (R$)

DESPESAS A PAGAR:
  - Nr emp.
  - Av. liquid. (aviso de liquidação)
  - Data
  - Credor/Fornecedor
  - Unidade orçamentária
  - Espécie
  - Tipo
  - Função
  - Valor(R$)

RETENÇÕES:
  - Sequência
  - Data
  - Nr emp.
  - Credor/Fornecedor
  - CNPJ
  - Retenção (tipo: INSS, ISS, IRRF)
  - Função
  - Tipo
  - Valor retido

--------------------------------------------------------------------------------
OBSERVAÇÕES IMPORTANTES
--------------------------------------------------------------------------------

1. SUBSTITUIÇÃO DE DADOS
   Ao importar novos dados, os dados anteriores do MESMO TIPO são substituídos.
   Exemplo: Se importar novos "Empenhados", apenas os Empenhados são substituídos.
   Os dados de Liquidados, Pagos, etc. permanecem inalterados.

2. ARQUIVO ÚNICO PARA TODAS AS DESPESAS
   Você pode criar um único arquivo Excel com várias abas:
   - Aba "Empenhados"
   - Aba "Liquidados"
   - Aba "Pagos"
   - Aba "A Pagar"
   - Aba "Retidos"
   O sistema identificará e importará cada uma automaticamente.

3. FORMATO DE DATA
   As datas podem estar nos formatos:
   - DD/MM/AAAA (02/01/2024)
   - AAAA-MM-DD (2024-01-02)
   O sistema converte automaticamente para DD/MM/AAAA na exibição.

4. FORMATO DE VALORES
   - Use ponto ou vírgula como separador decimal
   - Não use separador de milhares, ou use ponto
   - Exemplos válidos: 1234.56, 1234,56, 1234567.89

5. ARMAZENAMENTO LOCAL
   Os dados são armazenados no navegador (IndexedDB).
   - Cada navegador/computador tem seus próprios dados
   - Limpar dados do navegador apaga os dados importados
   - Mantenha backup dos arquivos Excel originais

--------------------------------------------------------------------------------
SOLUÇÃO DE PROBLEMAS
--------------------------------------------------------------------------------

PROBLEMA: Dados não aparecem após importação
SOLUÇÃO:
  - Verifique se selecionou o tipo correto (Despesas/Receitas)
  - Confirme se clicou em "Confirmar Importação"
  - Verifique se o nome da aba é reconhecível
  - Atualize a página (F5)

PROBLEMA: Erro ao importar
SOLUÇÃO:
  - Verifique se o arquivo é .xlsx (Excel)
  - Verifique se as colunas têm os nomes corretos
  - Tente com um arquivo menor para teste

PROBLEMA: Algumas abas não foram importadas
SOLUÇÃO:
  - Verifique se o nome da aba contém as palavras-chave
  - Renomeie a aba para algo mais reconhecível
  - Exemplo: "Dados" → "Empenhados"

--------------------------------------------------------------------------------
APÓS A IMPORTAÇÃO
--------------------------------------------------------------------------------

Depois de importar, acesse as páginas de consulta:

  Menu → Despesas → Empenhados     (para ver empenhos)
  Menu → Despesas → Liquidados    (para ver liquidações)
  Menu → Despesas → Pagos         (para ver pagamentos)
  Menu → Despesas → A Pagar       (para ver pendências)
  Menu → Despesas → Retidos       (para ver retenções)
  Menu → Despesas → Gerencial     (para dashboard)
  Menu → Despesas → Comparativos  (para comparar anos)

--------------------------------------------------------------------------------
EXPORTAÇÃO DE DADOS
--------------------------------------------------------------------------------

Nas páginas de consulta, você pode exportar os dados:

  - PDF: Gera documento formatado para impressão
  - Excel: Gera planilha .xlsx editável

A exportação considera:
  - Os filtros aplicados
  - A aba ativa (Resumo exporta agrupado, Detalhado exporta tudo)
